## Интерактивный дашборд онлайн-ритейла

Анализ транзакций интернет-магазина: выручка, активность клиентов, география продаж и возвраты. Построен на **Streamlit**, **Pandas**, **Plotly**.

---

### Описание
Дашборд помогает быстро увидеть:
- Где зарабатываем больше (страны, товары, клиенты)
- Как меняется выручка/возвраты во времени
- Какие клиенты активны и какие товары проблемные
- Быстрый экспорт отфильтрованных данных в CSV

### Стек и данные
- **Язык**: Python 3.9+
- **Фреймворк**: Streamlit
- **Библиотеки**: Pandas, NumPy, Plotly
- **Данные**: файл `Online_Retail.csv` (kaggle.com), ~540k транзакций за 2010–2011 гг.

---

### Быстрый старт (Streamlit)
1) Установите зависимости (при необходимости создайте venv):
   ```bash
   pip install -r requirements.txt
   ```
2) Запустите приложение:
   ```bash
   streamlit run app.py
   ```
3) Откройте ссылку из консоли (обычно `http://localhost:8501/`).

> Если данных нет, положите `Online_Retail.csv` в корень проекта (файл используется в коде по умолчанию).

### Структура основных функций
- `load_data()` — загрузка CSV с кэшем Streamlit
- `cleaning()` — очистка и подготовка данных (фильтрация аномалий, нормализация)
- `multiselect_with_all()` — мультиселекты с опцией «Все»
- `calculate_metrics()` — пересчёт KPI по отфильтрованным данным
- `create_all_charts()` — формирование всех графиков Plotly
- UI:
  - Главная (landing) с описанием и кнопкой «Запустить дашборд»
  - Фильтры в сайдбаре (даты, страны, клиенты, товары)
  - KPI-блок, вкладки с графиками, таблица + экспорт CSV
